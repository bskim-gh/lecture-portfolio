# 김보선의 강의 포트폴리오

## 📚 강사 소개
**김보선** - Python 프로그래밍 전문 강사

다양한 수준의 학습자를 대상으로 한 체계적인 Python 교육 경험을 보유하고 있으며, 실무 중심의 교육을 통해 학습자들의 실력을 향상시키는 것을 목표로 합니다.

## 🎯 강의 분야
- **Python 기초 프로그래밍**
- **객체지향 프로그래밍**
- **웹 크롤링**
- **Streamlit 웹 애플리케이션 개발**
- **비개발 부서를 위한 Python 교육**

## 📖 강의 커리큘럼

### 1. Python 기초 개념
- **대상**: Python 입문자
- **내용**: Python 기본 문법, 데이터 타입, 제어문, 함수 등
- **파일**: `Python/python_개념/`

### 2. Python 실습
- **대상**: 기초 문법을 익힌 학습자
- **내용**: 실무 중심의 Python 프로그래밍 실습
- **파일**: `Python/Python_실습/`

### 3. 객체지향 프로그래밍
- **대상**: Python 중급자
- **내용**: 클래스, 상속, 다형성, 고급 기능
- **파일**: `Python/Python_객체지향/`

### 4. 웹 크롤링
- **대상**: 데이터 수집에 관심 있는 학습자
- **내용**: 웹 스크래핑, 데이터 추출, API 활용
- **파일**: `Python/Python_웹크롤링/`

### 5. Streamlit 웹 애플리케이션
- **대상**: 웹 앱 개발에 관심 있는 학습자
- **내용**: Streamlit을 활용한 대시보드 및 웹 앱 개발
- **파일**: `Python/Python_streamlit/`

### 6. 비개발 부서용 Python
- **대상**: 비개발 직무자
- **내용**: 업무 자동화, 데이터 분석 기초
- **파일**: `Python/Python_비개발부서용/`

## 🛠️ 주요 기술 스택

- **BeautifulSoup, Selenium** - 웹 크롤링
